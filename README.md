# Zotero
An example using the Zotero Web API

## Installation

Install bower and npm dependencies:

```bash
$ npm install && bower install
```

## Configure

Create your own `.zoterorc` with your settings. For instance:

```yml
[zotero]
  user = 1019792
  key = msFCwEsfT6D2692qcP0dplyZ
  collection = MA22XTCQ
```
